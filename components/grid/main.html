Grid is a layout utility which helps in position Components in DOM a cakewalk. You can use a Two column grid as well as a Three column Grid.

Two Column Grid

You can place as many as items and grid will place them in subsequent rows of 2 column each Just add the classes "grid" and "grid-col-2" to the parent and all it's children will be equally divide into cells.
